一、产品概述

GPU iPavo 是 DCE 5.0 平台的 GPU 综合管理与 AI 运维仪表盘。面向平台管理员和运维人员，提供从基础设施健康到 AI 运维收益的全链路可视化监控能力。

核心用户画像与关注点

角色 关注重点 (Painpoints & Goals)
平台管理员 集群整体健康状态、资源容量规划、紧急告警处理
GPU 运维工程师 GPU 实时的计算利用率、显存分配效率、空闲分配导致的资源浪费
AI 平台负责人 AI ROI 投资回报率、模型 API 消费账单、安全防护与大模型护栏拦截效果
财务与成本管理 集群资源消费账单、各模型消费明细

二、页面功能模块总览

仪表盘由 7 个核心的独立数据模块组成，从系统基础状态层层递进到前沿的大模型 AI 成本与安全指标：

序号 模块名称 核心功能定位
1 基础运维概览 容器运行健康蜂窝图/仪表盘、CPU与内存历史趋势折线、集群组件就绪度判定及活跃告警
2 GPU 资源概览 快速展现安装了 GPU 的 K8s 节点与设备的总数就绪状态，尤其是“已分配但处于空闲”的关键浪费指标
3 GPU 利用率与集群统计 GPU分配率、计算利用率、显存分配率及实际使用率 4维半圆仪表盘；纳管集群的安装组件明细弹窗
4 资源用量 将传统计算资源(CPU/内存/Pods/磁盘)与 GPU 资源(卡数/显存)融合的 6维环形进度面板
5 GPU 利用率详情表 提供按“集群”与“节点”维度进行表格聚合的能力，支持时间窗口切换、5列指标排序与页面跳转
6 AI 智能运维 AI 调度带来的 ROI 累计金额、大模型安全护栏(拦截敏感信息/注入)次数趋势、实时安全评分及模型消费
7 集群消费情况 各集群按节点数、容器组数以及成本计量引擎核算出的绝对消费金额的排行明细

5.1 GPU 资源概览

展示平台纳管的 GPU 硬件底层物理存量。该模块最核心的设计指标是“空闲已分配 GPU 设备”，它是识别企业 AI 资源浪费与调度黑洞的核心利器。

空闲已分配 (Allocated but Idle) 定义: 工作负载在 YAML 中 Request 申请了 GPU 算力，但底层 DCGM 引擎监控到其实际 GPU Core Utilization 持续处于极低状态。此指标能帮助运维追踪占着物理卡却不跑任务的“占位”容器。

6.1 GPU 利用率四维指标与集群组件

四维仪表盘聚焦于 GPU 的核心配额和实际使用效率的对比，避免将“显存分完”误判为“算力耗光”。同时，集群卡片支持通过“更多”按钮查看详细的容器组件分布。

利用率维度 指标定义与公式 健康/低效阈值建议
1. GPU 分配率 已分配 GPU 卡数 / 平台总物理 GPU 数 < 90% 处于安全运行区间；超 95% 限制新 Pod 调度。
2. GPU 计算利用率 DCGM_FI_DEV_GPU_UTIL 核心硬件计算频率均值 > 50% 判定为 AI 算力使用高效，低于 15% 属于低效运行。
3. GPU 显存分配率 K8s Requests 显存总量 / 平台物理显存总量 < 90% 正常；因 AI 任务极易发生 OOM，此配额极度关键。
4. GPU 显存利用率 实际物理使用显存字节 / 已分配的显存总量 > 60% 为高效；通常用于核对训练任务中 Batch Size 的饱满度。

8.1 集群/节点 GPU 利用率详细表格

当管理员掌握了宏观指标后，需要进行切片下钻。此模块提供对集群和节点维度的深度列表透视，提供表格的多列排序、维度交互切换和深度数据链接。

排序与交互规范

  • 聚合维度切换 (Cluster ↔ Node): 下拉框切换，自动变动列头为“集群名称”或“节点名称”，并同步重算所有聚合值。
  • 时间窗口切换: 支持“1h / 24h / 7d”窗口选择。1小时展现短期尖峰，7天展示长期分配走势。
  • 排序表头 (5 列可排): GPU配额、平均已分配卡数、计算利用率、显存利用率及关联工作负载均支持多态点击（升序、降序、默认）。

9.1 AI 智能运维 (ROI/护栏/安全/模型消费)

AI 智能运维是 GPU iPavo 面向 AI 大模型时代的**王牌差异化模块**。将模型推理、护栏防御、大模型安全审计等抽象为四大高维财务与风控看板。

时间选择器联动规范 (Confirm 逻辑):
AI ROI 与护栏干预图表由**统一时间选择器**强力控制。触发时间变更时，为防止高频无效刷新，设计要求：**选定范围后必须手动点击“确认”才真正应用重绘**，同时具备一键刷新动作。
子指标卡 控制源 业务价值与计算模型
1. AI ROI 时间选择器 展示 AI 智能引擎（精细调度、自动切卡回收等）为平台累计节省的资金金额走势，呈面积上升累积趋势。
2. 护栏干预次数 时间选择器 展示 AI 护栏网关（在提示词注入攻击、幻觉拦截、PII敏感脱敏等）执行的紧急拦截次数序列。
3. AI 安全评分 实时快照 (不受控) 实时快照评分指标。由 4 子维度加权计算：提示词注入防御(90)、敏感信息覆盖(90)、内容安全(90)和幻觉拦截，提供安全建议摘要。
4. 模型 API 消费表 独立计量 支持对企业内部多款大模型（如 GPT-4o, DeepSeek, Qwen）调用产生的 API Token 及换算费用进行翻页和字段排序。
